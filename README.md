# find-difference-game
DxLibで制作した間違い探しゲームです。

ゲーム概要
画像の中に間違いをクリックして探すゲームです。
どれぐらい早く終わったかとか正答率もあります。

プレイ動画

制作意図
unityゲームジャムに投稿こうしたゲームをもとに、プログラミングやゲームエンジンを理解するためにDxlibで再実装しました。

工夫した点
ランダム配置にすることで毎回結果が変わる
場合によってはもうひとつが見えずらかったので一定時間でヒントを出すように

開発環境
C++
DxLib
Visual Studio
DxLib公式サイトからダウンロードしてプロジェクトに設定してください。
https://dxlib.xsrv.jp/dxdload.html

機能
タイトル画面
プレイ画面
リザルト画面
フェード画面
ランダム配置

操作方法
マウスクリックで間違いを選択

開発状況
ステージ３まで追加完了。クリアしたのちタイトルに戻るように
正解音と不正解音を追加
時間がたつとヒントを出すように
